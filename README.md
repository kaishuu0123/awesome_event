## Heroku Deploy 方法
 * 以下の環境変数を heroku の環境変数に設定してください
   * TWITTER_CONSUMER_KEY
   * TWITTER_CONSUMER_SECRET 

コマンド例　

```
$ heroku config --app <app_name> TWITTER_CONSUMER_KEY=<CONSUMER KEY>
$ heroku config --app <app_name> TWITTER_CONSUMER_SECRET=<CONSUMER SECRET>
```
